# European Stadiums Project

This repository contains analysis of new stadiums built in Europe over the last 20 years.

## Structure

- `analysis.ipynb` – main Jupyter notebook documenting the data analysis.
- `data/` – supporting datasets (.csv or .xlsx) used in the analysis.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/european-stadiums.git
   ```
2. Install dependencies (e.g., pandas, matplotlib, seaborn, etc.).
3. Open and run `analysis.ipynb` in Jupyter.

## Data

Place any CSV or Excel files with stadium information in the `data/` folder. Example dataset columns might include:

- `stadium_name`
- `city`
- `country`
- `opening_year`
- `capacity`
